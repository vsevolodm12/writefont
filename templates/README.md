# Шаблоны для Calligraphr

Эта папка должна содержать 7 шаблонов для создания шрифта:

1. **template_cyrillic_1.png** - Шаблон для русских букв (часть 1)
2. **template_cyrillic_2.png** - Шаблон для русских букв (часть 2)
3. **template_cyrillic_3.png** - Шаблон для русских букв (часть 3)
4. **template_latin_1.png** - Шаблон для английских букв (часть 1)
5. **template_latin_2.png** - Шаблон для английских букв (часть 2)
6. **template_symbols_1.png** - Шаблон для специальных символов (часть 1)
7. **template_symbols_2.png** - Шаблон для специальных символов (часть 2)

## Как получить шаблоны

1. Перейдите на https://www.calligraphr.com/
2. Зарегистрируйтесь или войдите
3. Перейдите в раздел "Download Templates"
4. Скачайте шаблоны для:
   - Кириллицы (3 шаблона)
   - Латиницы (2 шаблона)
   - Специальных символов (2 шаблона)
5. Переименуйте файлы согласно списку выше
6. Поместите их в папку `templates/`

## Формат файлов

Файлы должны быть в формате PNG или JPG. Рекомендуется использовать PNG для лучшего качества.
